# Test Instructions: Vocabulary Library (生词库)

这些测试说明**与框架无关**。

## 概述

生词库功能允许用户查看、搜索、过滤和管理所有保存的单词。

---

## 用户流程测试

### 流程 1: 浏览生词列表

**场景:** 用户查看已保存的所有单词

#### 成功路径

**步骤:**
1. 用户点击底部导航"生词库"图标
2. 页面显示单词列表（列表视图）
3. 每个单词卡片显示：单词、音标、释义、发音按钮
4. 用户可以滚动查看更多单词

**预期结果:**
- [ ] 列表按添加时间倒序排列（最新的在前）
- [ ] 显示总单词数量
- [ ] 每个单词卡片布局整齐
- [ ] 滚动流畅，无性能问题

#### 切换视图

**步骤:**
1. 用户点击"网格视图"切换按钮

**预期结果:**
- [ ] 切换到网格布局（2-3列）
- [ ] 单词卡片以网格形式排列
- [ ] 切换按钮状态更新

---

### 流程 2: 搜索和过滤单词

**场景:** 用户使用搜索和标签功能查找特定单词

#### 搜索功能

**步骤:**
1. 用户在搜索框输入 `serendipity`
2. 实时显示搜索结果

**预期结果:**
- [ ] 只显示匹配的单词
- [ ] 搜索匹配单词拼写或中文释义
- [ ] 显示匹配的单词数量
- [ ] 清空搜索框后恢复显示所有单词

#### 标签过滤

**步骤:**
1. 用户点击"工作词汇"标签

**预期结果:**
- [ ] 只显示带有该标签的单词
- [ ] 选中的标签高亮显示
- [ ] 可以选择多个标签（AND逻辑）

#### 排序切换

**步骤:**
1. 用户切换排序方式为"字母顺序"

**预期结果:**
- [ ] 列表按A-Z重新排序
- [ ] 排序方式显示在筛选器中

---

### 流程 3: 查看单词详情

**场景:** 用户点击单词查看完整信息

**步骤:**
1. 用户点击列表中的某个单词
2. 导航到单词详情页面
3. 页面显示单词完整信息：拼写、音标、释义、例句、学习记录
4. 用户可以点击发音按钮

**预期结果:**
- [ ] 显示单词的所有信息
- [ ] 显示例句（英文+中文翻译）
- [ ] 显示学习记录：添加日期、复习次数、掌握程度
- [ ] 提供"开始练习"按钮
- [ ] 提供"删除单词"按钮

---

### 流程 4: 删除单词

**场景:** 用户从生词库中删除单词

**步骤:**
1. 用户在单词卡片上点击删除图标
2. 显示确认对话框："确定要删除这个单词吗？"
3. 用户点击"确定"

**预期结果:**
- [ ] 单词从列表中移除
- [ ] 显示成功提示："单词已删除"
- [ ] 列表更新（空状态或显示剩余单词）

---

## 空状态测试

### 无单词状态

**场景:** 用户尚未保存任何单词

**预期结果:**
- [ ] 显示空状态插图或图标
- [ ] 显示标题："还没有生词"
- [ ] 显示描述："拍照识别单词，开始积累词汇"
- [ ] 显示主CTA按钮："去拍照"
- [ ] 点击按钮导航到拍照页面

### 搜索无结果

**场景:** 用户搜索不存在的单词

**预期结果:**
- [ ] 显示消息："未找到匹配的单词"
- [ ] 显示建议："尝试其他搜索词或清除筛选"
- [ ] 提供"清除筛选"链接

### 标签过滤无结果

**场景:** 用户选择某个标签但没有匹配的单词

**预期结果:**
- [ ] 显示消息："该标签下没有单词"
- [ ] 显示建议："选择其他标签或添加单词到该标签"

## 边界情况

- [ ] 处理超过100个单词的列表（性能测试）
- [ ] 处理非常长的单词或释义（文本截断）
- [ ] 处理搜索特殊字符
- [ ] 处理网络错误加载失败
- [ ] 从空到有数据：首次保存单词后列表更新
- [ ] 从有数据到空：删除最后一个单词
