# Foundation Specification

## Overview
基础模块提供用户认证、应用导航和系统基础设施功能。包括用户注册、登录、忘记密码、以及新用户引导流程，为整个应用提供稳定的底层支撑。

## User Flows
- **新用户注册流程**：输入邮箱/手机号 → 设置密码 → 验证码验证 → 完成注册 → 进入引导页
- **用户登录流程**：输入邮箱/手机号 → 输入密码 → 登录 → 进入首页
- **忘记密码流程**：输入注册邮箱/手机号 → 验证码验证 → 设置新密码 → 完成重置
- **新用户引导流程**：欢迎页 → 功能介绍（3-4页滑动） → 选择学习偏好 → 完成引导 → 进入拍照页面

## UI Requirements

### 登录页面
- 邮箱/手机号输入框
- 密码输入框（带显示/隐藏切换）
- "忘记密码？"链接
- "登录"主按钮
- "还没有账号？立即注册"链接
- 社交账号登录选项（可选）
- 保持登录状态开关

### 注册页面
- 邮箱/手机号输入框
- 验证码输入框（带发送验证码按钮，60秒倒计时）
- 密码输入框（带强度提示）
- 确认密码输入框
- 用户协议和隐私政策复选框
- "注册"主按钮
- "已有账号？立即登录"链接

### 忘记密码页面
- 邮箱/手机号输入框
- 验证码输入框
- 新密码输入框
- 确认新密码输入框
- "重置密码"主按钮
- "返回登录"链接

### 引导页面
- 全屏滑动视图（3-4页）
- **欢迎页**：应用Logo、欢迎标语、开始按钮
- **功能介绍页1**：拍照识别功能演示动画 + 说明文字
- **功能介绍页2**：生词库和练习功能演示动画 + 说明文字
- **功能介绍页3**：学习统计功能演示动画 + 说明文字
- **偏好设置页**：选择当前英语水平（初级/中级/高级）、每日学习目标
- 分页指示器（小圆点）
- "跳过"和"下一步/开始使用"按钮

### 底部导航
- **首页**（拍照图标）- 默认首页
- **生词库**（书本图标）
- **练习**（笔图标）
- **我的**（用户图标）
- 选中状态有高亮显示
- 未选中状态灰色显示

## Technical Requirements
- 支持邮箱和手机号两种登录方式
- 密码加密传输和存储
- JWT Token 认证机制
- 自动刷新 Token
- 本地持久化登录状态
- 引导页只在首次登录后显示一次
