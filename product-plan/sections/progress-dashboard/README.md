# Progress Dashboard Specification

## Overview
学习数据可视化中心，帮助用户全面了解学习进展。通过直观的图表和统计数据，展示生词掌握情况、练习表现、复习进度和学习成就，激励用户持续学习。

## User Flows
- 用户进入进度统计页面，看到学习概览（今日/本周/本月统计数据）
- 向下滚动查看各类数据可视化图表（学习活动趋势、掌握程度分布、正确率曲线、生词增长）
- 查看生词统计详情（总词汇量、掌握程度分布、标签统计）
- 浏览学习历程（最近学习记录、练习历史时间线）
- 查看已获得的成就徽章和即将解锁的里程碑
- 点击图表或卡片可查看详细数据（如某一天的练习记录）

## UI Requirements
- 顶部学习概览卡片（今日/本周/本月的关键指标）
- 学习活动趋势图（折线图显示学习天数和练习次数）
- 掌握程度分布图（环形图或堆叠条显示已掌握/熟悉/学习中的比例）
- 正确率曲线图（折线图显示每日练习正确率变化）
- 生词增长曲线图（面积图显示累计生词数量增长）
- 生词统计卡片（总词汇量、各掌握程度数量、热门标签）
- 学习历程时间线（最近学习记录和练习历史）
- 成就徽章展示区（已获得和即将解锁的里程碑）
- 移动端响应式设计
- Light/Dark 模式支持
