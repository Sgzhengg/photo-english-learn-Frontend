# Test Instructions: Progress Dashboard (进度统计)

这些测试说明**与框架无关**。

## 概述

进度统计页面通过图表和数据可视化展示用户的学习进展和成就。

---

## 用户流程测试

### 流程 1: 查看学习概览

**场景:** 用户查看整体学习统计

**步骤:**
1. 用户点击底部导航"统计"图标
2. 页面显示学习概览卡片
3. 显示今日/本周/本月的统计数据

**预期结果:**
- [ ] 显示生词总数（如：256个）
- [ ] 显示今日新增单词（如：8个）
- [ ] 显示今日练习次数（如：3次）
- [ ] 显示今日正确率（如：85%）
- [ ] 显示连续学习天数（如：7天）

---

### 流程 2: 查看学习活动趋势

**场景:** 用户查看学习活动的时间趋势

**步骤:**
1. 用户向下滚动到"学习活动趋势"图表
2. 显示折线图或柱状图
3. X轴显示日期（最近7天或30天）
4. Y轴显示学习次数或新增单词数

**预期结果:**
- [ ] 图表数据准确反映学习活动
- [ ] 鼠标悬停（或点击）显示具体数值
- [ ] 可以切换时间范围（7天/30天/90天）
- [ ] 图表有清晰的标题和图例

---

### 流程 3: 查看掌握程度分布

**场景:** 用户查看单词掌握情况

**步骤:**
1. 用户查看"掌握程度分布"图表
2. 显示环形图或堆叠条形图
3. 分类显示：已掌握/熟悉/学习中/未学习

**预期结果:**
- [ ] 显示各掌握程度的单词数量
- [ ] 显示百分比分布
- [ ] 使用不同颜色区分程度
- [ ] 点击某分类可以查看该类别的单词列表

---

### 流程 4: 查看学习成就

**场景:** 用户查看已获得的成就徽章

**步骤:**
1. 用户滚动到"成就徽章"区域
2. 显示已获得的徽章网格
3. 显示即将解锁的徽章（灰度显示）

**预期结果:**
- [ ] 已获得的徽章彩色显示
- [ ] 未获得的徽章灰度显示
- [ ] 点击徽章显示解锁条件
- [ ] 显示解锁进度（如：15/20）

---

### 流程 5: 查看详细数据

**场景:** 用户点击图表查看详细数据

**步骤:**
1. 用户点击某一天的柱状图
2. 显示该日的详细学习记录

**预期结果:**
- [ ] 弹出详细信息卡片
- [ ] 显示日期：2025年1月15日
- [ ] 显示新增单词：10个
- [ ] 显示练习次数：5次
- [ ] 显示复习单词：25个
- [ ] 显示正确率：90%
- [ ] 提供"关闭"按钮

---

## 空状态测试

### 无学习数据

**场景:** 用户首次使用，还没有学习记录

**预期结果:**
- [ ] 显示友好插图
- [ ] 显示标题："开始您的学习之旅"
- [ ] 显示描述："完成第一次学习后，这里会显示您的进度"
- [ ] 显示主CTA按钮："开始学习"
- [ ] 点击按钮导航到拍照页面

### 某项数据为空

**场景:** 用户有生词但还没有练习记录

**预期结果:**
- [ ] 概览卡片显示生词数量
- [ ] 练习相关统计显示 "暂无数据"
- [ ] 不显示空图表，而是显示提示消息

---

## 组件交互测试

### ProgressDashboard

**渲染正确:**
- [ ] 显示所有概览卡片
- [ ] 显示所有图表
- [ ] 显示成就徽章
- [ ] 显示学习历程时间线

**用户交互:**
- [ ] 点击图表查看详细信息
- [ ] 切换时间范围（7天/30天）
- [ ] 点击成就徽章查看详情
- [ ] 点击"查看更多"链接跳转到相关页面

**数据刷新:**
- [ ] 下拉刷新数据
- [ ] 显示加载状态
- [ ] 刷新后更新所有图表

---

## 边界情况

- [ ] 处理大量数据（1000+单词）的图表性能
- [ ] 处理零值数据（某天无学习记录）
- [ ] 处理极端数值（100%正确率或0%正确率）
- [ ] 处理长时间跨度（365天）的数据显示
- [ ] 处理网络错误加载数据失败
- [ ] 从空到有数据：首次学习后的数据更新

---

## 图表渲染测试

### 学习活动趋势图
- [ ] X轴标签正确显示日期
- [ ] Y轴刻度合理
- [ ] 数据点准确
- [ ] 折线或柱状图样式正确

### 掌握程度分布图
- [ ] 环形图或饼图比例正确
- [ ] 颜色区分清晰
- [ ] 图例正确

### 正确率曲线图
- [ ] 数据点连线平滑
- [ ] 显示趋势
- [ ] 标注峰值和谷值

## 注意事项

- 测试图表在不同屏幕尺寸下的响应式布局
- 验证数据计算的准确性（总数、平均数、百分比）
- 测试图表的交互性能（悬停、点击）
- 确保数据加载时有骨架屏或加载指示器
- 测试从零到有数据的状态转换动画
