# Practice & Review Specification

## Overview
通过智能间隔重复算法，为用户安排每日练习任务。提供多种练习类型（填空、中译英选择、听写默写），实时反馈和进度追踪，帮助用户科学高效地巩固生词。

## User Flows
- 用户进入练习页面，看到今日任务卡片（显示需复习的单词数量和练习类型）
- 用户点击开始练习，进入练习流程
- 系统根据智能算法推送需要复习的单词
- 用户完成练习（填空/选择/听写），获得即时反馈（正确/错误提示、正确答案）
- 练习结束后显示结果总结（得分、正确率、用时）
- 答错的单词自动加入错题复习队列
- 系统记录学习进度，安排下次复习时间（基于艾宾浩斯遗忘曲线）
- 用户可以查看复习日程和进度统计

## UI Requirements
- 今日任务卡片（显示单词数量、练习类型、预计用时）
- 练习类型选择（填空/中译英/听写默写）
- 练习题目展示区域（根据题型显示不同界面）
- 即时反馈区域（正确/错误提示、正确答案显示）
- 进度指示器（当前题号/总题数）
- 结果总结页面（得分、正确率、用时统计）
- 错题复习入口
- 复习日程视图（显示下次复习时间、复习队列）
- 进度统计图表（学习天数、掌握程度趋势）
- 复习提醒通知（应用内徽章、通知提醒）
- 移动端响应式设计
- Light/Dark 模式支持
