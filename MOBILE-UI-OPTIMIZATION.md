## 🔑 如何退出登录

### 方法 1：通过用户菜单（推荐）
1. 访问：`http://localhost:5173/`
2. 点击右上角的**用户头像**（圆形按钮）
3. 在下拉菜单中点击**"登出"**（红色按钮）
4. ✅ 自动跳转到登录页面

### 方法 2：清除浏览器数据
1. 按 `F12` 打开开发者工具
2. 进入 **Application** 标签
3. 左侧找到 **Local Storage** → `http://localhost:5173`
4. 删除以下 keys：
   - `access_token`
   - `refresh_token`
   - `photoenglish_mock_current_user`
5. 刷新页面

### 方法 3：浏览器控制台
1. 按 `F12` 打开控制台
2. 输入以下命令：
```javascript
localStorage.clear()
location.reload()
```

## 🐛 如果还是看不到登录页

### 检查是否已登录
打开浏览器控制台（F12），输入：
```javascript
console.log(localStorage.getItem('access_token'))
```

- 如果输出 token 字符串 → 已登录，需要退出
- 如果输出 null → 未登录

### 强制退出
复制以下代码到控制台：
```javascript
localStorage.clear()
sessionStorage.clear()
location.href = '/login'
```
