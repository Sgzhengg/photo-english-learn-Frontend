# Vocabulary Library Specification

## Overview
用户可以查看和管理从拍照识别中保存的所有生词。支持列表和网格两种视图切换，提供基础搜索和标签分类功能，每个单词可以查看详情、播放发音、开始练习或删除。

## User Flows
- 用户进入生词库，默认显示列表视图，展示所有保存的单词
- 用户可以切换到网格视图，以卡片形式浏览单词
- 用户可以通过搜索框搜索单词拼写或中文释义
- 用户可以通过标签筛选单词（预设标签如"工作词汇"、"考试必备"等，也可以自定义标签）
- 用户可以切换排序方式（添加时间、学习进度、字母顺序等）
- 用户可以在单词卡片上快速操作：播放发音、查看详情、开始练习、删除单词
- 用户点击单词进入详情页，查看基础信息、例句和学习记录

## UI Requirements
- 顶部搜索栏
- 视图切换按钮（列表/网格）
- 标签筛选器（预设标签 + 自定义标签）
- 排序方式切换器
- 单词列表视图和网格视图
- 单词卡片包含：单词、音标、释义、发音按钮、快捷操作按钮
- 单词详情页面：基础信息、例句展示、学习记录（添加日期、复习次数等）
- 移动端响应式设计
- Light/Dark 模式支持
