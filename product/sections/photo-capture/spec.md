# Photo Capture Specification

## Overview
用户通过拍照识别图片中的主要物体，系统显示中英文名称列表。用户可以选择感兴趣的单词保存到生词库，同时系统基于图片场景生成英语描述句子并提供 TTS 朗读功能，朗读时对应单词高亮显示，帮助用户学习语境中的词汇使用。

## User Flows
- 用户点击拍照按钮进入相机取景器
- 拍照后，AI 识别图片中的主要物体，显示中英文名称列表
- 用户点击单词旁的保存按钮，将单词保存到生词库
- 保存后，页面下方自动显示基于图片场景生成的英语描述句子
- 用户点击播放按钮，TTS 朗读整个句子
- 朗读过程中，当前读到的单词在句子中高亮显示，跟随语音进度

## UI Requirements
- 相机取景器界面（取景框、拍照按钮、闪光灯控制）
- 识别结果页面：单词卡片列表
- 单词卡片包含：英文单词、中文释义、音标、发音播放按钮、保存按钮
- 场景句子显示区域（位于结果页面下方）
- 句子播放按钮和语音进度高亮效果
- 无需历史记录功能
