# PhotoEnglish（拍照学英语）

## Description
PhotoEnglish 是一款面向英语爱好者的移动端学习应用，通过拍照识别技术从英文阅读材料中快速提取生词，并提供语境化练习和间隔重复复习系统，帮助用户从"认识单词"进阶到"会用单词"。

## Problems & Solutions

### Problem 1: 查过的单词只会认、不会用
传统查词方式只提供释义，用户知道意思但不会在真实场景中使用。PhotoEnglish 提供 AI 生成的语境化练习句子，让用户在真实场景中掌握单词用法。

### Problem 2: 手动记录生词效率低下
遇到生词需要逐个查词典、手动记录，过程繁琐耗时。通过拍照批量识别 + 智能提取技术，一键保存多个生词，大幅提升积累效率。

### Problem 3: 缺乏系统的复习机制
查过的单词没有科学复习计划，很快就会遗忘。基于间隔重复算法的默写复习系统，自动调整复习频率，巩固长期记忆。

### Problem 4: 学习进度难以量化
用户不清楚自己学了多少、掌握程度如何。可视化统计面板清晰展示生词数量、练习正确率、复习进度等关键指标。

## Key Features
- 📸 智能拍照识别：批量提取英文单词，即时显示释义，快速勾选保存
- 📚 生词库管理：标签化整理，搜索过滤，随时查看
- ✍️ 语境练习：AI 生成包含生词的练习句子，填空/翻译训练
- 🔄 间隔复习：默写模式检测记忆效果，自动调整复习频率
- 📊 学习统计：生词数量、练习正确率、复习进度一目了然
- 🔊 TTS 发音：地道英语发音，边听边学
